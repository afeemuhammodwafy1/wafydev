# 🚀 WafyOps - Developer & Utility Toolbox

WafyOps is a premium, privacy-focused, and fully responsive client-side utility suite designed to simplify daily digital and development tasks. Built entirely with vanilla web technologies, it features a modern, dark-themed glassmorphic user interface with fluid micro-interactions.

## ✨ Key Features

*   **🔒 Password Generator:** Generate highly secure, customized passwords with adjustable lengths and strict toggle criteria (Uppercase, Lowercase, Numbers, Special Characters).
*   **🔤 Text Case & Analytics:** Perform instant text case mutations (Upper, Lower, Title) and get real-time character and word count statistics.
*   **📷 Live QR Code Generator:** Convert URLs, credentials, or standard text into customized visual QR tags dynamically with instant download capability as `wafyops-qr.png`.
*   **🎨 Color Palette Terminal:** Inspect visual hex/rgb colors dynamically using an optimized circular color-wheel picker component.
*   **📱 100% Responsive Design:** Smooth, edge-to-edge adaptive layouts engineered perfectly for both wide desktop environments and mobile viewports.
*   **⚡ One-Click Workflows:** Fast performance utilizing purely client-side memory with zero external database tracking.

---

## 🛠️ Tech Stack Used

*   **HTML5** – Structured semantic layout and components.
*   **CSS3** – Advanced Grid/Flexbox layouts, Custom Media Queries, and Glassmorphism styling.
*   **JavaScript (ES6+)** – Client-side cryptographic functions, DOM Manipulation, and Tab-switching workflow logic.
*   **Font Awesome** – Modern and crisp vector iconography.
